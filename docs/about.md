# About RecNextEval

## Origin and Purpose

RecNextEval is an open-source Python toolkit developed as a post project after
[streamsight](https://github.com/HiIAmTzeKean/streamsight) was developed in
Nanyang Technological University (NTU), Singapore. The project was taken
under the supervision of Professor Sun Aixin.
